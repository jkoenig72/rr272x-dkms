rr272x-dkms

This is the packaging for the HighPoint RocketRaid 2710-2722 models. The original driver can be obtained on the HighPoint website. The driver is updated to the not yet officially available 1.6.1 release and should work with all newer kernels >3.8 just as well

The only part of the driver which is actually open-source is the kernel interface. Everything else is just blobs. So it can't really be considered OSS and it's unlikely that it will ever be included in the debian package tree. Here is a package repo for the driver. In case there will ever be updates I'll update that one too.

wget -O - http://p.l42.at/signing-pub.key | sudo apt-key add -
echo "deb http://p.l42.at/debian wheezy main" | sudo tee /etc/apt/sources.list.d/rr272x.list
sudo apt-get update && sudo apt-get install rr272x-dkms
